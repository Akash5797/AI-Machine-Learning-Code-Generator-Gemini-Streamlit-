# AI Machine Learning Code Generator (Gemini + Streamlit)

An **AI-powered Machine Learning code generator** built using **Streamlit**, **SmolAgents**, and **Google Gemini (via LiteLLM)**.

This app allows users to describe a Machine Learning task in **natural language**, and the AI automatically generates a **complete, production-ready Python ML pipeline**.

---

## Features

- Describe ML tasks in plain English
- AI-powered ML code generation
- Uses Google Gemini models via LiteLLM
- Generates end-to-end ML pipelines
- Supports sklearn, pandas, matplotlib, seaborn
- Web search capability via DuckDuckGo
- Secure API key handling with `.env`
- Clean UI with output persistence
- Code returned in Markdown (`python`)
